## 编译命令行
-	安装依赖：npm install
-	启动调试：npm run dev 
-	编译压缩：npm run publish

# 注意
* 每次要在全局添加样式或者脚本，要改动public/index.html时，需要运行一次npm run dll，重新编译那些被排除在热加载之外的js文件。
* 样式采用less。
* 已经配置了hot loader、router 4.0和redux。
