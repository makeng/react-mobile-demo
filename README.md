# React Demo (Vite based)

## About

个人/公司的 React 项目的快速搭建。

所有的框架、配置都是最新的，以便快速实验各种特性。

```
# Vite required
pnpm install
pnpm run dev
```



## 已包含的主题（同时是路由）

### /test-redux

跨页面如何用「临时 model」或者「全局变量」取数据，能最少程度地污染全局？



### /test-auth

如何在系统中，全局地加载「权限数据」，以实现权限的控制？



### /test-functional-update

「函数式更新」与普通的 setState 更新有何不同？到底解决了什么问题？
